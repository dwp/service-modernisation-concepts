Future State Design Service Modernisation Programme Prototype

Purpose of the MVP

Explore journeys within the end-to-end service blueprint across the three benefits within MVP scope. To see if it is possible to create common data, process and service delivery model.

The three benefits in scope are:

  • Attendance Allowance
  • Carer's Allowance
  • Disability Living Allowance Child

Versions

Version 1

The first MVP based on a basic flow for a user to log in and report a change of caring circumstances.

  • Persona - Joanna
  • Journey flow includes log in to account, navigate to a change of caring responsibilities, answer flow of quesitons, confirmation screen with information about the change to entitlement
  • Secondary flow included later - linked profile
